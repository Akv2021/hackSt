Commands

Enable logging
adb shell setprop log.tag.FA VERBOSE
adb shell setprop log.tag.FA-SVC VERBOSE


Enable debug View
adb shell setprop debug.firebase.analytics.app com.riviera.indifi.app

Disable debug view
adb shell setprop debug.firebase.analytics.app .none.
OR
Reboot the device
